## What
正如Rainsho在lesson107中说的，React是一个View层解决方案，并不适合大型应用，处理复杂的组件间通信会变的一团乱麻。所以2014年Facebook提出了Flux架构的概念，而Redux就是
Flux的一个社区实现，发布后很短时间内就成为了最热门的前端框架。
